Getting Started

4D Timetree is a visual editor for multi-world, branching timelines — built for tabletop RPGs, speculative fiction, and any story where time isn't simple.

1 Set the year range below — this defines the span of your diagram's horizontal axis.
2 Add timelines (Lines tab) — one per world, dimension, or character path. Each gets its own color and row.
3 Place events — use Markers for named story beats and Nodes for silent reference points.
4 Connect timelines — use Welds for bridges between worlds and Intersections where timelines converge or end.
5 Add narrative arcs — use Causal Links for conditional story branches and Branches for visual forks.
6 Export your diagram as SVG or JSON using the toolbar above.

Timelines

A timeline is a horizontal row representing a world, era, or character path. Each has its own color and can optionally diverge from another timeline at a specific year.

Timelines
Nodes

A node is an unmarked point on a timeline — no label, just a connection anchor. Use nodes when you need a precise reference point for welds or causal links without cluttering the diagram. Click a bead on the canvas to edit, or add one below.

Nodes
Markers

A marker is a named, visible event on a timeline — shown with a label, optional category icon, and clickable annotation. Use markers for story beats, key encounters, and any event that needs a narrative description or causal link. Click a bead on the canvas to edit, or add one below.

Markers
Welds

A weld draws a vertical bridge between two timeline points — typically at the same year, to show two worlds sharing a moment. Use welds for time-travel arrivals, portal crossings, or any two-world convergence.

Editing weld
Welds
Intersections

An intersection marks where multiple timelines converge at a single year. Timelines can optionally terminate here — useful for collapses, endings, or merges where a world ceases to exist independently.

Branches

A branch draws a simple curved arc between any two timeline points — useful for divergences, splits, or visual connections not tied to named events. For conditional narrative connections between events, use Causal Links instead.

Editing branch
Branches
Wormholes

A wormhole draws repeating sine-wave arcs from an anchor year to one or more destination years on the same timeline. Use wormholes for time loops, recursive travel, or any cyclical connection within a single world.

Editing wormhole
Wormholes
Causal Links

A causal link connects two named events with a conditional arc — "if X, then Y follows." To draw one, click two event beads on the canvas while on this tab. Click any existing arc to edit its criteria and notes.

All Links
Cycles

A cycle draws a dashed vertical line at a year — useful for recurring events, seasonal patterns, or any moment that repeats identically across all timelines.
