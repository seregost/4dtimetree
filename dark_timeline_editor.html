Schema reference

worlds: array of timelines. Each has id, label, color, row (vertical slot, 0=top).

yearRange: [startYear, endYear] defines the x-axis time span.

intersections: array of convergence points. Each has year and worlds (list of world ids that meet there). Worlds bend toward a shared y at this year.

wormholes: array of intra-world wormholes. Each has world, anchor (year), jumps (list of years), side ("above"/"below"), optional window: { open, close } (ISO date strings). Without window: point connection, dot endpoints. With window: temporal window, bracket endpoints (I-beam). The window dates apply simultaneously at the anchor and all jump years. Anchor marker is suppressed when it coincides with an intersection.

welds: array of cross-world bridges. Each has from: {world, year}, to: {world, year}, optional label.

branches: array of new timelines that emerge from another. Each has fromWorld, fromYear, newWorld (id of the resulting world, must exist in worlds), and the new world's first year is set by its branch point.

worlds[].divergeFrom: optional { world, year }. When set, the world's path begins at the source world's y-position at that year and curves asymptotically down to its own row, modeling a fork rather than an independent start.

nodes: array of single-world stable events (e.g. God Particles). Each has world, year, optional label. Renders identically to an intersection (amber dot + dashed ring) but on a single worldline without bending others. Wormhole endpoint markers are suppressed when a node occupies the same position.

segments: array of highlighted time spans on a worldline. Each has world, from, to (years), optional label, optional labelSide ("above"/"below", default "below"). Renders as a semi-transparent thick band over the worldline with an italic label — use for periods where significant time passes or characters age.

markers: array of named events. Each has world, year, label, optional color.
